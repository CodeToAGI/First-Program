# CodeToAGI - Episode 2 Challenge
## Verify Installation & Write Your First Program

### 🎯 Challenge Description
After installing Python, verify it works by writing and running your first Python program with 3 print statements.

**Steps:**
1. Open terminal/command prompt
2. Verify Python: `python --version` (Windows) or `python3 --version` (Mac/Linux)
3. Create `hello.py` file
4. Write 3 `print()` statements
5. Run: `python hello.py`

### 📝 Code (hello.py)
```python
print("Python is installed!")
print("I am ready to learn!")
print("Let us build AI together!")
